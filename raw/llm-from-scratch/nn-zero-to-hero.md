## Description

A comprehensive course on neural networks presented as a series of YouTube videos with accompanying Jupyter notebooks. The course begins with foundational concepts and progressively builds toward modern deep learning architectures. It "starts all the way at the basics" and assumes "basic knowledge of Python and a vague recollection of calculus."

## Course Lectures

**Lecture 1: Micrograd — Backpropagation Fundamentals**
Covers backpropagation and neural network training through building a minimal automatic differentiation engine.

**Lecture 2: Makemore Part 1 — Bigram Language Models**
Introduces PyTorch tensors and character-level language modeling using bigrams, establishing frameworks for training, sampling, and loss evaluation.

**Lecture 3: Makemore Part 2 — Multilayer Perceptrons**
Implements MLPs for character-level modeling while covering essential machine learning concepts including hyperparameter tuning, train/test splits, and overfitting.

**Lecture 4: Makemore Part 3 — Activations, Gradients & BatchNorm**
Examines internal mechanics of deep networks, diagnostic visualization tools, and introduces Batch Normalization as a key innovation for training stability.

**Lecture 5: Makemore Part 4 — Manual Backpropagation**
Implements backpropagation without autograd, building intuition about gradient flow through compute graphs.

**Lecture 6: Makemore Part 5 — WaveNet Architecture**
Develops convolutional neural networks using hierarchical tree structures, demonstrating practical deep learning development workflows.

**Lecture 7: Building GPT from Scratch**
Constructs a Generatively Pretrained Transformer following the "Attention Is All You Need" paper and GPT-2/3 architectures.

**Lecture 8: GPT Tokenizer**
Covers the tokenization pipeline using Byte Pair Encoding, addressing common LLM quirks stemming from tokenizer design.
